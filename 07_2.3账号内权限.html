2.3 Permissions within the account

View on Yuque
Special reminder: This function is only available with [Merchant version token]. If you are a user, please use 4.6 to obtain permission information
Permissions within the account
Interface description
To log in to the merchant version platform with an account, please use this interface to obtain the internal permissions of the account under the merchant. Use this to determine what operations the account can perform on the power station and equipment.
Request URL
https://api.solarmanpv.com/account/v1.0/role
