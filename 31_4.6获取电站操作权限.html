4.6 Obtain power station operation permissions

View on Yuque

Special reminder: This function is only available with [User Version Token]. If you are a merchant, please use 2.3 to obtain account permissions
Obtain power station operation permissions
Interface description
To log in to the user version platform with an account, please use this interface to obtain the account's operating permissions for the power station and its equipment. When there is no permission for some operations of the power plant, an error is reported when calling the interface.

https://api.solarmanpv.com/station/v1.0/role
