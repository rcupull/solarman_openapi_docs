4.3 Obtain historical data of power station

View on Yuque

Obtain historical data of power plant
Interface description
Obtain historical data of the power station and support interval data query of frame, day, month and year.
Request URL
https://api.solarmanpv.com/station/v1.0/history
Request method
post
Request type
application/json
Return type
*/*

2 (daily dimension): Statistical data for a specified number of days (no more than 30 days); there is power generation level statistical data;
3 (month dimension): Specify statistical data for several months (no more than 12 months)
4 (year dimension): Specify statistical data for several years
