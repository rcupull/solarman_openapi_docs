4.18 Query real-time weather

View on Yuque
According to the power station area ID (for details, see10 Dictionary Information-Region) can query the real-time weather conditions of the power station. The real-time weather is updated every 3 hours. It is recommended that the call interval for a single power station is 3 hours.
