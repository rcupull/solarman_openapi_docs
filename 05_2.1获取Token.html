Get token
Interface description
Token is the credential for calling resources. Currently, it supports the mobile phone number, email address, and user name of the registered account. You can choose one of three to log in (mobile, email, and username fields must be passed. When the mobile field is passed, the countryCode field must be passed). If the orgId is not passed, the C-side user Token will be obtained. When the orgId value is passed, the Token of the corresponding merchant will be obtained.
